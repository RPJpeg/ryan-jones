Services & Skills

I have experience in both graphic design and web development. Here are the services I can offer you.

Customizable

I can build websites that are unique to your project or business. Whether it be your fonts, colors, or ideal layout, I can modify the attributes to best for your needs.

Responsive

All the sites that I create will adjust to provide the best possible appearance based on what device is being used to view it.

Graphics Support

Using Graphic Design as one of my core foundations, I can easily create custom icons, images, logos and color schemes that help your website look professional.

Clean Code

I make a point to create sites that are extremely organized, easy to update and have the cleanest code possible.

Modern Design

Using the newest trends and tools, I can create a site that looks new and fresh. I can use these styles to really get your site to stand out.

Web Hosting

If you're looking for someone to take care of your web hosting, look no further! I would be happy to research and choose the best service that will work best for your site and save you money.

Portfolio

Some examples of my web design work. These are either samples that I have put together to show what I can create for you, or examples of work that I have created for some of my previous clients.

Preston Wealth Management

This page was created for Preston Wealth Managment, a company stationed here in the DC Metro area. This site was used with a combination of HTML, CSS and Javascript/jQuery. The Bootstrap framework was utilized in the creation.

A fixed header, scroll spy, and modals were a few things used to enhance the easy of navigation. The site is fully responsive, working well on large screens as well as mobile devices. Finally, a contact form was created, to be able to receive information from their potential clients.

Created around December 2015

Kitten Crew

This sample page was created shortly after I completed my class on Front-end Web Development at General Assembly. I used it to show that I am comfortable using image grids and buttons. I also demonstrated that I can effectively use image carousels. This is a good example page of something I could create that uses a login or sign up page, when paired with some back-end development.

Created around October 2015

Placeholder Inc.

This is another page I created to get some practice in. This is a relatively simple page, but I think it does a good job of showing how sometimes simplicity can go a long way. I think this page looks clean, professional and is easy to navigate. As always, it is mobile responsive!

Created around November 2015

UX Mountain

I created this mock website to illustrate some of the more current trends and processes that I have learned. I used SASS and wrote the entire thing from scratch. The only framework or library I used on this project was Font Awesome, for some quick icons to use for some of the content. I am very pleased with the end result on this one.

Created November 2016

Resume Template

This site was created to allow a potential user to input their work history and professional skills into this template to have a highly customized online presence. It is responsive, clean and there are many moving parts that help it stand our from a standard profile. This can be easily linked to a LinkedIn profile or PDF resume.

Created October 2016

Mount Vernon Rehabilitation

This website was created for Mount Vernon Rehabilitation Medicine Associates. I focused on creating a very user friendly experience, since the target audience for this website would vary. It's clean and easy to navigate. I was especially pleased with the tabbed feature in the Contact page for the different locations.

Created August 2016
